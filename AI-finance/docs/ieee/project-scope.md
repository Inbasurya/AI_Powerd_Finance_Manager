# FinMind AI Pro - Research Scope Notes

## Intended IEEE Demonstration Tracks

- Intelligent personal finance analysis with explainable ML
- Hybrid anomaly and forecasting pipelines
- User-centric decision support (savings and event planning)

## Suggested Evaluation Metrics

- Anomaly detection: precision, recall, F1, ROC-AUC
- Forecasting: MAE, RMSE, MAPE
- Financial scoring calibration: correlation with expert labels
- Explainability quality: SHAP consistency analysis

## Reproducibility Checklist

- Fixed random seeds in model training
- Version-pinned dependencies
- Dataset schema validation
- Train/test split protocol documentation
